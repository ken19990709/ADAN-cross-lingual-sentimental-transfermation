# ADAN-cross-lingual-sentimental-transfermation
此项目旨在使用源语言带情感标签预料与目标语言无情感标签语料进行跨语言情感迁移实验。本实验使用了GAN的思想与ADAN模型来实现迁移。
实验目标为得到一个能够较好对两种语言进行情感分类的模型。最终实验结果如下
